# JavaScript 真假值

#### 作者
noone

#### 日期
2017-10-09

#### 标签
   javascript , 数据结构

---
|数值类型|转换成布尔值|
|---|---|
|undefined  |  false|
|null  |  false  |
|布尔值  |  true是 true，false是false  |
|数字  |  +0、-0和 NaN都是false，其他都是true|
|字符串  |  如果字符串是空的（长度是0）就是 false，其他都是true  |
|对象   |  true|
