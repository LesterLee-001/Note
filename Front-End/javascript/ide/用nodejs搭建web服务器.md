## 安装http-server
```
sudo npm install http-server -g
```
## 在工作路径中输入http-server
```
Leslie:javascript-datastructures-algorithms noone$ http-server
```
> 通过访问`localhost:8080` 查看
> 通过`ctrl+c`退出http-server
