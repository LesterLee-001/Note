### 步骤
1. 安装node版本管理模块`n`

```
sudo npm install -g n
```

2. 升级node.js到最新稳定版

```
n stable
```

3. `n`后面也可以跟随版本号比如：
```
n v0.10.26
```

或
```
n 0.10.26
```
